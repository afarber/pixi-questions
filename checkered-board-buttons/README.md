# Checkered Board Buttons

8x8 checkered board with draggable colored tiles, custom UI components, and audio feedback.

![screenshot](https://raw.github.com/afarber/pixi-questions/master/checkered-board-buttons/screenshot.gif)

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build production version
- `npm run test:run` - Run tests
- `npm run lint` - Run ESLint

## Custom UI Components

- **Button** - States: default, hover, pressed, disabled
- **Checkbox** - Simple checkbox with checkmark
- **SectionedList** - Scrollable container with scrollbar
- **ScrollBar** - Standalone scrollbar
