# drag-tiles-layout

Drag-and-drop tile game with 3D effects using PixiJS Layout.

![screenshot](https://raw.github.com/afarber/pixi-questions/master/drag-tiles-layout/screenshot.gif)

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build production version
- `npm run test:run` - Run tests
- `npm run lint` - Run ESLint

## Features

- 8x8 checkered board with grid snapping
- 3D perspective mesh transformations
- Dynamic shadows on drag
- Localization (English, German, French)
